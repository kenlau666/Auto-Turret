# Auto-Targeting Turret

## 1. Introduction
This is an embedded system project with four main functions
1. Auto-Targeting
2. Vertical and Horizontal Rotation
3. Shooting
4. Manual Controlling

## 2. Components
1. STM32 Development Borad
2. OV7725 (camera)
3. Two Motors
4. Three Servos 
5. LCD
6. L293D
7. Resistors
8. Buttons
